# Backend

This folder contains the Laravel backend for the project.

The Laravel PHP application now lives here, so the backend code is organized in
the standard Laravel folders:

- `app/` for controllers, models, and providers
- `routes/` for web and API routes
- `database/` for migrations, seeders, and factories
- `resources/views/` for Blade templates
- `config/`, `bootstrap/`, `public/`, `storage/`, and `tests/`

Typical backend commands run from this folder:

```bash
composer install
php artisan serve
```
